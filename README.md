A plain-HTML/CSS/JS To-Do app with 6 themes + light mode. Good for portfolios and internship tasks.

## Features
- 6 themes: Professional, Colorful, Hacker, Card, Dashboard, Student
- Light mode toggle
- Add / Edit (inline) / Delete / Toggle done
- Filters: All / Active / Completed
- Search
- Import / Export (JSON)
- LocalStorage persistence
- Keyboard shortcuts: Enter to add, E to edit first, Del to delete focused, Ctrl+S to export

## Run
Open `index.html` in your browser. Or open any `todo-*.html` file to view a standalone theme.

## Extend ideas
- Add due dates and reminders (local notifications or backend)
- Add drag-and-drop order persistence
- Hook to Firebase for multi-device sync
